How to Run Minecraft

Windows

Create a file named run.bat in the extracted folder with:

@echo off
java -Xmx2G -Xms1G -jar client.jar

Double-click run.bat to start the game.

Linux/Mac

Create a file named run.sh in the extracted folder with:

#!/bin/bash
java -Xmx2G -Xms1G -jar client.jar

Make it executable and run:

chmod +x run.sh
./run.sh

Universal

Create a file named run.txt with:

java -Xmx2G -Xms1G -jar client.jar

Save as run.bat (Windows) or run.sh (Linux/Mac)

Notes:

  • Make sure you have Java installed (Java 8 or newer)
  • Adjust -Xmx2G and -Xms1G based on your RAM:
    • 2GB RAM: -Xmx1G -Xms512M
    • 4GB RAM: -Xmx2G -Xms1G
    • 8GB RAM: -Xmx4G -Xms2G
  • You need a valid Minecraft account to play
