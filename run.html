How to Run Minecraft

Prerequisites

  • Java 8 or newer installed on your system
  • A valid Minecraft account
  • The downloaded Minecraft version files (from the downloader)

Windows

  1. Extract the downloaded ZIP file to a folder of your choice
  2. Create a new file named run.bat in the extracted folder
  3. Open the file with a text editor and paste this code:
    @echo off
    java -Xmx2G -Xms1G -jar client.jar
  4. Save the file and double-click run.bat to start the game

Linux/Mac

  1. Extract the downloaded ZIP file to a folder of your choice
  2. Create a new file named run.sh in the extracted folder
  3. Open the file with a text editor and paste this code:
    #!/bin/bash
    java -Xmx2G -Xms1G -jar client.jar
  4. Make the file executable:
    chmod +x run.sh
  5. Run the game:
    ./run.sh

RAM Allocation

Adjust the -Xmx and -Xms values based on your system's RAM:

System RAM Recommended Settings Command Example
2GB -Xmx1G -Xms512M java -Xmx1G -Xms512M -jar client.jar
4GB -Xmx2G -Xms1G java -Xmx2G -Xms1G -jar client.jar
8GB -Xmx4G -Xms2G java -Xmx4G -Xms2G -jar client.jar

Troubleshooting

Common Issues:

  • Java not found: Make sure Java is installed and added to your system's PATH
  • Out of memory: Reduce the -Xmx value if you get memory errors
  • Game won't start: Try running the command from a terminal/command prompt to see error messages
